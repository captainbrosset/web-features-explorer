Picture-in-picture (video)

The picture-in-picture API allow websites to create a floating, always-on-top video window. Also known as PiP or pop-out video.

Compatibility

  • Chrome 69
  • Chrome on Android 105
  • Edge 79
  • Firefox ❌
  • Firefox on Android ❌
  • Safari 13.1
  • Safari on iOS 13.4
