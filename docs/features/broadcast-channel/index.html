BroadcastChannel

The BroadcastChannel API allows you to send messages between same-origin browsing contexts, such as between the same page loaded in multiple tabs.

Compatibility

  • Chrome 60
  • Chrome on Android 60
  • Edge 79
  • Firefox 57
  • Firefox on Android 57
  • Safari 15.4
  • Safari on iOS 15.4
