3D transforms

The transform CSS property and its 3D transform functions allow rotations and other transforms in three dimensions, including perspective transforms.

Compatibility

  • Chrome 36
  • Chrome on Android 36
  • Edge 12
  • Firefox 16
  • Firefox on Android 16
  • Safari 15.4
  • Safari on iOS 15.4
