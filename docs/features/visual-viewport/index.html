Visual viewport API

The visualViewport API provides a way to query and modify the user-visible viewport of a web page.

Compatibility

  • Chrome 61
  • Chrome on Android 61
  • Edge 79
  • Firefox 91
  • Firefox on Android 68
  • Safari 13
  • Safari on iOS 13
