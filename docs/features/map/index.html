Map (initial support)

Map objects hold key-value pairs and remember the original insertion order of the keys.

  • Chrome 38
  • Chrome on Android 38
  • Edge 12
  • Firefox 13
  • Firefox on Android 14
  • Safari 8
  • Safari on iOS 8
