Web animations

The web animations API allows you to animate and synchronize the animations of DOM elements.

Compatibility

  • Chrome 84
  • Chrome on Android 84
  • Edge 84
  • Firefox 80
  • Firefox on Android 80
  • Safari 16
  • Safari on iOS 16
