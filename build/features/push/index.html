Push messages

The Push API subscribes to and receives server-initiated messages. Subscribers receive pushed messages in the background, even after periods inactive or offline.

Compatibility

  • Chrome 50
  • Chrome on Android 50
  • Edge 17
  • Firefox 44
  • Firefox on Android 48
  • Safari 16
  • Safari on iOS 16.4
