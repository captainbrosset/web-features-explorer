Constraint validation API

Methods that validate form controls before submission, such as checkValidity(), reportValidity() and setCustomValidity().

Compatibility

  • Chrome 40
  • Chrome on Android 40
  • Edge 18
  • Firefox 51
  • Firefox on Android 64
  • Safari 10.1
  • Safari on iOS 10.3
