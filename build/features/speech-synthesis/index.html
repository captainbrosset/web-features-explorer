Speech synthesis

The SpeechSynthesis API converts text into audio using a synthetic voice.

Compatibility

  • Chrome 33
  • Chrome on Android 33
  • Edge 14
  • Firefox 49
  • Firefox on Android 62
  • Safari 7
  • Safari on iOS 7
